P Problema Inicial

Resolvamos: √3(√3 + 5)

🎯 Paso 1: Entender el problema

Tenemos una raíz cuadrada multiplicando un paréntesis. Usaremos la propiedad distributiva.

💡 Paso 2: Sustituir para facilitar
Si √3 = a

Entonces: √3(√3 + 5) = a(a + 5)
🔢 Paso 3: Propiedad distributiva
a(a + 5) = a × a + a × 5 = a² + 5a
🔄 Paso 4: Regresar a la raíz
a² + 5a = (√3)² + 5(√3) = 3 + 5√3
⚡ Recuerda: (√3)² = 3 porque la raíz y el cuadrado se cancelan
✅ Resultado Final
3 + 5√3

C Concepto Clave

📚 Propiedad Distributiva

Cuando multiplicas un número por un paréntesis, multiplicas por CADA término:

Fórmula:

a(b + c) = ab + ac
🌟 Con raíces funciona igual:

√5(√6 + √5) = √5×√6 + √5×√5 = √30 + 5
  • ✅ Multiplica el número de fuera por CADA término del paréntesis
  • ✅ √a × a = a (se cancelan)
  • ✅ √a × √b = √(ab)

E Ejemplo Paso a Paso

Resolvamos: √5(√45 + 7)

🔍 Paso 1: Simplificar √45
√45 = √(9×5) = √9 × 5 = 3√5
📝 Paso 2: Reescribir
√5(3√5 + 7)
⚡ Paso 3: Distribuir
√5 × 3√5 + √5 × 7
🧮 Paso 4: Resolver
3(√5)² + 7√5 = 3×5 + 7√5 = 15 + 7√5
🎉 Resultado Final
15 + 7√5
