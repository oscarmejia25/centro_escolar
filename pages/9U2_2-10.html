P Problema Inicial

Vamos a resolver: \(\sqrt{3}(\sqrt{3} + 5)\)

🎯 Paso 1: Entender el problema

Tenemos una raíz cuadrada multiplicando un paréntesis. Esto es como tener:

"3 raíces de 3, multiplicando a (3 raíces de 3 más 5)"

💡 Paso 2: Usar una letra para simplificar

Para hacerlo más fácil, vamos a llamarle "a" a \(\sqrt{3}\):

Si \(\sqrt{3} = a\)

Entonces: \(\sqrt{3}(\sqrt{3} + 5) = a(a + 5)\)
💭 Piensa así: Es como si tuviéramos "a" cajas con "a + 5" dulces cada una. ¿Cuántos dulces hay en total?
🔢 Paso 3: Aplicar la propiedad distributiva

La propiedad distributiva dice que multiplicamos "a" por cada cosa dentro del paréntesis:

\(a(a + 5) = a \times a + a \times 5 = a^2 + 5a\)
🔄 Paso 4: Regresar a la raíz cuadrada

Ahora regresamos \(\sqrt{3}\) en lugar de "a":

\(a^2 + 5a = (\sqrt{3})^2 + 5(\sqrt{3})\)
⚡ Dato importante: Cuando elevas al cuadrado una raíz cuadrada, ¡se cancelan!
\((\sqrt{3})^2 = 3\)
✅ Paso 5: Resultado final
\((\sqrt{3})^2 + 5\sqrt{3} = 3 + 5\sqrt{3}\)

¡Listo! La respuesta es: \(\mathbf{3 + 5\sqrt{3}}\)

C Concepto Clave

📚 La Propiedad Distributiva

Imagina que tienes 3 bolsas, y cada bolsa tiene 2 manzanas y 4 naranjas.

¿Cuántas frutas tienes en total?

\(3(2 + 4) = 3 \times 2 + 3 \times 4 = 6 + 12 = 18\) frutas

🎓 Fórmula General:

Para cualquier número real \(a\), \(b\) y \(c\):

\(\mathbf{a(b + c) = ab + ac}\)
🌟 Con raíces cuadradas funciona igual:

\(\sqrt{5}(\sqrt{6} + \sqrt{5}) = \sqrt{5} \times \sqrt{6} + \sqrt{5} \times \sqrt{5}\)

\(= \sqrt{30} + 5\)
  • ✅ La propiedad distributiva SIEMPRE funciona con números reales
  • ✅ Funciona perfecto con raíces cuadradas
  • ✅ Solo multiplica el número de fuera por CADA término del paréntesis

E Ejemplo Paso a Paso

Resolvamos: \(\sqrt{5}(\sqrt{45} + 7)\)

🔍 Paso 1: Simplificar la raíz si se puede

Antes de distribuir, revisamos si \(\sqrt{45}\) se puede simplificar:

\(\sqrt{45} = \sqrt{9 \times 5} = \sqrt{9} \times \sqrt{5} = 3\sqrt{5}\)
💡 Tip: Busca cuadrados perfectos dentro de la raíz (4, 9, 16, 25, 36, 49...)
📝 Paso 2: Reescribir el ejercicio
\(\sqrt{5}(\sqrt{45} + 7) = \sqrt{5}(3\sqrt{5} + 7)\)
⚡ Paso 3: Aplicar propiedad distributiva

Multiplicamos \(\sqrt{5}\) por cada término:

\(\sqrt{5} \times 3\sqrt{5} + \sqrt{5} \times 7\)
🧮 Paso 4: Resolver cada multiplicación

Primer término: \(\sqrt{5} \times 3\sqrt{5} = 3 \times (\sqrt{5})^2 = 3 \times 5 = 15\)

Segundo término: \(\sqrt{5} \times 7 = 7\sqrt{5}\)

🎉 Paso 5: Resultado final
\(15 + 7\sqrt{5}\)
